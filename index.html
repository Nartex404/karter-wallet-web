Política de Privacidad

1. Introducción

Karter Wallet ("la Aplicación") respeta su privacidad. Esta Política de Privacidad explica cómo recopilamos, usamos y protegemos sus datos. La Aplicación es gratuita y es mantenida gracias a la publicidad.

2. Recopilación y Uso de Datos

La Aplicación utiliza servicios de terceros que pueden recopilar información utilizada para identificarlo.

Datos Financieros
Todos los datos financieros (ingresos, gastos, presupuestos) se almacenan de forma segura en Google Cloud Firestore bajo su propia cuenta de usuario autenticada. No vendemos ni compartimos sus datos financieros con terceros.

Permisos Solicitados

  • Micrófono (RECORD_AUDIO): Se utiliza EXCLUSIVAMENTE cuando usted toca el botón de micrófono para transcribir sus gastos mediante voz. El audio se procesa para convertirlo a texto y no se almacena permanentemente.
  • Biometría (FaceID / TouchID): Se utiliza localmente en su dispositivo para asegurar el acceso a la aplicación. No recopilamos sus datos biométricos.
  • Identificadores de Dispositivo: Utilizamos identificadores para mostrar publicidad personalizada y análisis de estabilidad.

3. Servicios de Terceros

La Aplicación utiliza los siguientes servicios de terceros, los cuales tienen sus propias Políticas de Privacidad:

  • Google AdMob: Para mostrar anuncios. AdMob puede usar identificadores de publicidad para mostrar anuncios relevantes. Política de Privacidad de Google.
  • Firebase Analytics & Crashlytics: Para recopilar datos anónimos sobre bloqueos y uso de la aplicación con el fin de mejorar la estabilidad.
  • Google Gemini AI: Para procesar consultas del Asesor Financiero. El texto de sus consultas se envía a Google para su procesamiento, pero no se utiliza para entrenar modelos públicos sin su consentimiento explícito.

4. Seguridad de los Datos (Data Safety)

Valoramos su confianza al proporcionarnos sus datos personales y nos esforzamos por utilizar medios comercialmente aceptables para protegerlos. Sin embargo, recuerde que ningún método de transmisión a través de Internet o método de almacenamiento electrónico es 100% seguro y confiable.

5. Eliminación de Datos

Usted tiene derecho a solicitar la eliminación de sus datos. Puede hacerlo directamente desde la aplicación en Ajustes > Cuenta > Eliminar Cuenta, lo cual borrará permanentemente sus registros de nuestros servidores.

6. Privacidad de los Niños

Estos Servicios no se dirigen a nadie menor de 13 años. No recopilamos a sabiendas información de identificación personal de niños menores de 13 años.

7. Cambios a esta Política

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Se le aconseja revisar esta página periódicamente para ver si hay cambios.
